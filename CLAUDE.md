# 波妞记账app — 产品文档

---

## 一、项目概述

### 1.1 产品定位

**波妞记账app** 是一款面向个人/家庭日常记账的 iOS 应用，以粉色为主色调，提供快速记账、自然语言解析、资产管理、年度统计等功能，帮助用户轻松掌握个人财务健康状况。

### 1.2 目标平台

| 项目 | 规格 |
|------|------|
| 设备 | iPhone 12 Pro Max |
| 屏幕尺寸 | 6.7 英寸 |
| 分辨率 | 1284 × 2778 像素（@3x） |
| 安全区域 | 顶部 47pt（状态栏），底部 34pt（Home Indicator） |
| 系统版本 | iOS 16.6 |
| 最低部署目标 | iOS 16.0 |

### 1.3 核心功能一览

1. **记账模块** — 手动选择收入/支出分类（一级科目 → 二级科目）记账；支持自然语言文本框输入，自动解析分类与金额
2. **资产管家** — 管理现金、储蓄卡、信用卡、虚拟账户、负债、自定义类目资产，支持添加/编辑/删除账户
3. **统计图表** — 按年份展示资产/负债/净资产趋势；按月度/年度查看各一级科目下二级科目的消费总金额（分类钻取）；支持收入同理
4. **首页概览** — 当月收支汇总、最近交易记录、净资产快览；支持月份左右滑动切换浏览历史月份
5. **搜索筛选** — 按关键词、分类、日期范围快速查找交易记录
6. **CSV 导入** — 支持从 CSV 文件批量导入历史记账数据，自动匹配分类
7. **CSV 导出** — 将交易数据导出为 CSV 文件，用于备份或在电脑上分析

## 六、自然语言解析引擎（TransactionParser）

### 6.1 设计原理

解析引擎采用**规则匹配 + 关键词库**的方式，无需网络请求，本地完成，保障用户隐私。

### 6.2 解析流程

```
用户输入文本
    │
    ▼
┌──────────────────┐
│ 1. 分词          │  按标点（逗号、句号、空格、换行）拆分为句子
│    拆分句子       │
└──────┬───────────┘
       │
       ▼
┌──────────────────┐
│ 2. 逐句解析       │  对每个句子：
│    提取关键信息    │  - 提取金额（正则匹配 "数字+元/块/¥"）
│    - 提取金额     │  - 提取关键词（匹配分类关键词库）
│    - 匹配分类     │  - 提取日期（如"今天"、"昨天"、"7月22日"）
│    - 提取日期     │
└──────┬───────────┘
       │
       ▼
┌──────────────────┐
│ 3. 组装结果       │  生成 Transaction 数组，返回预览
│    返回预览列表    │
└──────────────────┘
```
